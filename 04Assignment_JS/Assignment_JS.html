*JAVASCRIPT BASIC & DOM*

1. What is JavaScript?

JavaScript is a versatile programming language primarily used for web development. It allows developers to add interactivity, dynamic content, and behavior to websites. Originally created to make web pages interactive, JavaScript has evolved into a powerful language used for both client-side and server-side development.


2. What is the use of isNaN function?

In JavaScript NaN is short for "Not-a-Number".
The isNaN() method returns true if a value is NaN.
The isNaN() method converts the value to a number before testing it.


3. What is negative Infinity?

NEGATIVE_INFINITY is a property of the JavaScript Number object. It's a numeric value in JavaScript that signifies a value that is lower than any other number, including negative numbers.
You can only use it as Number.NEGATIVE_INFINITY.

5. What are undeclared and undefined variables?

Undeclared variables are those that have not been declared or defined in the current scope, while undefined variables are those that have been declared but not given a value.

7. What is the difference between ViewState and SessionState?

ViewState SessionState
A ViewState is a state of a page within a browser wherein the values of controls persist when post back operation is done. SessionState is the data of a user session and is maintained on the server side.
If the user requests another page, the data of previous page is no longer available. The availability of the user data is up to the completion of a session or closure of the browser.

8. What is === operator?

The === operator in JavaScript is the strict equality operator. It compares two values, checking not only if they are equal in value but also if they are of the same data type returning a Boolean result.


9. What are all the looping structures in JavaScript?

Loops offer a quick and easy way to do something repeatedly.
JavaScript supports several looping structures:

  1. for Loop
  2. while Loop
  3. do...while Loop
  4. for...in Loop
  5. for...of Loop

10. How can you convert the string of any base to an integer in JavaScript?

In JavaScript, you can convert a string representation of a number in different bases (binary, octal, hexadecimal) to an integer using the parseInt() function. This function takes the string to parse as the first argument and the base as the second argument.


11. What is the function of the delete operator?

The delete operator in JavaScript is used to remove a property from an object. It allows you to delete both an object's own properties and properties inherited from its prototype chain.

Syntax: delete object[property]


12. What are all the types of Pop up boxes available in JavaScript?

In Javascript, popup boxes are used to display the message or notification to the user.
There are three types of pop-up boxes in JavaScript:

1. Alert Box:

It is used when a warning message is needed to be produced. When the alert box is displayed to the user, the user needs to press ok and proceed.

2. Confirm Box

It is a type of pop-up box that is used to get authorization or permission from the user. The user has to press the ok or cancel button to proceed.

3.Prompt Box

It displays a message along with an input field where the user can enter some text. It's used to get input from the user.


13. What is the use of Void (0)?

The void operator evaluates the given expression and then returns undefined. In JavaScript, void(0) is a way to create an expression that evaluates to undefined. The void operator in JavaScript takes an operand and evaluates it, then returns undefined.
